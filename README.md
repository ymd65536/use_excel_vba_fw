# はじめに

このリポジトリは Excel VBA を使いやすくする UseVBAFrameWork のリポジトリです。
要はオレオレフレーム

## UseAccess

Microsoft Access をイイカンジに利用できるようにしてくれるモジュール

## UseCopy

クリップボードの利用をイイカンジにしてくれるモジュール

## UseExcel

Microsoft Excel VBA で利用できる機能をイイカンジに利用できるようにしてくれるモジュール

## UseFileSystem

ファイルシステムオブジェクトをイイカンジに利用できるにしてくれるモジュール

## UseHttp

非同期通信モジュール

## UseListObject

Microsoft Excel のリストオブジェクトを手軽に利用できるようにしてくれるモジュール

## UseReferConf

参照設定を bas ファイルにしてくれるモジュール

## UseString

文字列処理をイイカンジにやってくれるモジュール

## UseWMI

Windows Media Interface を利用したモジュール
VBA で MAC アドレスなどが取得できるようになる。

### UseWin32API

Windows32API が利用できるようになるモジュール
